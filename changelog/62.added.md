Prometheus intent lifecycle metrics in the Temporal worker (`/metrics` on `WORKER_METRICS_PORT`, default 9464): connectivity outcomes and provisioning duration emitted from the deployment activities, binding failures from the Infrahub fetch activity, plus the orphaned-rules and lineage-completeness gauge contract. Prometheus scrapes the worker via the new `synapse_worker` job, and the OrphanedRulesIncreasing / LineageCoverageDrop alerts now query these metric names.
